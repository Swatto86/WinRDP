üíº Why I Built This

I got tired of typing passwords over and over when connecting to multiple servers. So I built WinRDP to make my life easier:

  • ‚úì Encrypted storage - your host data is encrypted with Windows DPAPI, credentials go in Windows Credential Manager
  • ‚úì Set it and forget it - save your credentials once, never type them again
  • ‚úì One-click connections - double-click a host and you're connected, no prompts
  • ‚úì Smart defaults - use the same credentials everywhere, or override for specific servers
  • ‚úì Everything in one place - all your servers in a simple, searchable list
  • ‚úì Quick access anywhere - hit Ctrl+Shift+R to pop up WinRDP from whatever you're doing
  • ‚úì Always ready - starts with Windows so it's there when you need it

The result? I can connect to any of my servers in seconds. No more hunting for passwords, no more typing hostnames, no more sticky notes on my monitor.
