WinRDP Application Icon
=======================

The WinRDP icon is a professional blue monitor with a vibrant green frame.

Current Icon (app.ico):
-----------------------
- Multi-resolution ICO file (16x16 to 256x256)
- Blue screen representing Remote Desktop
- Green bezel/frame for visual distinction
- Professional monitor stand design
- Modern, clean appearance that scales well

How to Regenerate the Icon:
---------------------------

The icon is created using the included PowerShell script:

1. Run: .\create_icon.ps1
2. The script generates:
   - src\app.ico (multi-resolution ICO file)
   - src\app_icon_preview.png (256x256 preview)
3. Rebuild the application: .\build.bat

Customizing the Icon:
---------------------

To change the icon design:

1. Edit create_icon.ps1
2. Modify colors in the New-RDPIcon function:
   - $primaryBlue = Screen color
   - $greenBezel = Frame/bezel color
3. Adjust dimensions or proportions as desired
4. Run .\create_icon.ps1 to regenerate
5. Rebuild the application

The icon is automatically embedded in the executable via resources.rc:
  IDI_MAINICON ICON "app.ico"

Icon appears in:
- Window title bars
- System tray notification area  
- Taskbar
- Alt+Tab switcher
- File Explorer
- Installer
